# Customer Segmentation Analysis

## Project Overview
This project analyzes customer behavior and purchasing patterns using Power BI.

## Objectives
- Segment customers into High, Medium, and Low Value groups
- Analyze purchasing behavior
- Identify top-performing product categories
- Generate business recommendations

## Tools Used
- Microsoft Excel
- SQL
- Power BI

## Key Insights
- Clothing generated the highest revenue (104K)
- Low Value customers account for 38.82% of total customers
- Fall season recorded the highest purchases
- Only 27% customers are subscribed

## Business Recommendations
- Increase subscription adoption through loyalty programs
- Run targeted campaigns for low-value customers
- Focus marketing on high-performing categories
